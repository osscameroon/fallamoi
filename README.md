## FALLA-MOI

For a specific link, you can get somebody position and device informations

## REQUIREMENTS

- python (3.x recommended)
- pip3

## HOW TO INSTALL

```
# either using
make install

# or with pip directly
pip install -r requirements.txt
```

## HOW TO LAUNCH

```
# with make
make run
# or with python
python -m main
```
